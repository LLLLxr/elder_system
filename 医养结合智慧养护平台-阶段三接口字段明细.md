# 医养结合智慧养护平台 - 阶段三接口字段明细

> 范围：护理员每日服务执行/打卡、护士护理记录、医生查房记录、家属查看。

---

# 一、caredelivery 域接口

## 1. 护理员查询我的服务任务
### `GET /care-delivery/my-tasks`

### Query 参数
- `taskDate`：string，必填，格式 `YYYY-MM-DD`
- `elderId`：long，可选

### Response
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "elderId": 10001,
      "elderName": "王某某",
      "servicePlanId": 8101,
      "taskDate": "2026-05-04",
      "taskItems": [
        {
          "itemCode": "MEAL",
          "itemName": "送餐",
          "completed": false
        }
      ]
    }
  ]
}
```

---

## 2. 护理员提交服务打卡
### `POST /care-delivery/my-tasks/{servicePlanId}/check-in`

### Request Body
```json
{
  "elderId": 10001,
  "taskDate": "2026-05-04",
  "taskItems": [
    {
      "itemCode": "MEAL",
      "itemName": "送餐",
      "completed": true
    },
    {
      "itemCode": "CLEAN",
      "itemName": "清洁护理",
      "completed": true
    }
  ]
}
```

### Response
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "checkInRecordId": 9101,
    "completionStatus": "COMPLETED",
    "completionTime": "2026-05-04T09:00:00"
  }
}
```

---
