# 医养结合智慧养护平台 - 阶段四接口字段明细

> 范围：培训视频、季度考核、家属满意度、护理员综合考评。

---

# 一、quality 域接口

## 1. 医护端新增培训视频
### `POST /quality/training-videos`

### Request Body
```json
{
  "title": "基础护理操作规范",
  "videoUrl": "https://file.example.com/training-001.mp4",
  "durationSeconds": 1200,
  "enabled": true
}
```

### Response
- 返回 `trainingVideoId`

---

## 2. 医护端查询培训视频列表
### `GET /quality/training-videos`

### Query 参数
- `page`
- `size`
- `enabled`
- `title`

### 列表字段
- `trainingVideoId`
- `title`
- `videoUrl`
- `durationSeconds`
- `enabled`
- `createdAt`

---

## 3. 医护端启停培训视频
### `POST /quality/training-videos/{id}/toggle`

### Request Body
```json
{
  "enabled": false
}
```

---

## 4. 护理员端查询培训视频列表
### `GET /quality/training-videos/available`

### Response 列表字段
- `trainingVideoId`
- `title`
- `videoUrl`
- `durationSeconds`
- `progressPercent`
- `completed`

---

## 5. 护理员端更新培训播放进度
### `POST /quality/training-records/{videoId}/progress`

### Request Body
```json
{
  "progressPercent": 80,
  "completed": false
}
```

### Response
- 返回当前进度与完成状态

---

## 6. 护理员端查看我的培训记录
### `GET /quality/training-records/my`

### Query 参数
- `page`
- `size`
- `completed`

### 列表字段
- `recordId`
- `trainingVideoId`
- `title`
- `progressPercent`
- `completed`
- `completedAt`

---

## 7. 医护端发起季度考核
### `POST /quality/exams`

### Request Body
```json
{
  "quarterLabel": "2026-Q2",
  "caregiverIds": [5001, 5002],
  "questionnaireData": {
    "questions": [
      { "code": "Q1", "title": "护理流程是否规范" }
    ]
  }
}
```

### Response
- 返回创建成功数量或 exam 批次信息

---

## 8. 医护端查询季度考核列表

### Query 参数
- `page`
- `size`

### 列表字段

- `caregiverId`
- `caregiverName`
- `quarterLabel`
- `resultStatus`
- `score`
- `reviewedBy`
- `reviewedAt`

---

## 9. 医护端录入/审核考核结果

  "elderId": 10001,

  "servicePeriodEnd": "2026-05-31",

## 14. 生成综合考评结果
### `POST /quality/performance-summaries/generate`

### Request Body
```json
{
  "periodLabel": "2026-Q2",
  "caregiverId": 5001
}
```

### Response
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "summaryId": 9901,
    "caregiverId": 5001,
    "periodLabel": "2026-Q2",
    "examScore": 88,
    "familySatisfactionScore": 90,
    "overallScore": 89,
    "resultLevel": "GOOD"
  }
}
```

---

- `FAMILY_SATISFACTION_SURVEY`

## 2. 查询质量流程日志
### `GET /care-orchestration/workflow-transition-logs`

沿用前面结构。
