# 医养结合智慧养护平台 - 阶段五接口字段明细

> 范围：设备预警、风险待办、云探视、通知。

---

# 一、ops 域接口

## 1. 接收设备预警
### `POST /ops/device-alerts`

### Request Body
```json
{
  "elderId": 10001,
  "deviceId": "WB-001",
  "alertType": "HEART_RATE_HIGH",
  "alertLevel": "HIGH",
  "alertContent": "心率异常升高",
  "triggeredAt": "2026-05-04T11:00:00"
}
```

### Response
- 返回 `alertId`

---

## 2. 查询预警列表
### `GET /ops/device-alerts`

### Query 参数
- `page`
- `size`
- `elderId`
- `alertLevel`
- `alertStatus`

### 列表字段
- `alertId`
- `elderId`
- `deviceId`
- `alertType`
- `alertLevel`
- `alertContent`
- `alertStatus`
- `triggeredAt`
- `handledAt`
- `handledBy`

---

## 3. 查询预警详情
### `GET /ops/device-alerts/{id}`

### Response 字段
- `alertId`
- `elderId`
- `deviceId`
- `alertType`
- `alertLevel`
- `alertContent`
- `alertStatus`
- `triggeredAt`
- `handledAt`
- `handledBy`
- `handleResult`

---

## 4. 处理预警
### `POST /ops/device-alerts/{id}/handle`

### Request Body
```json
{
  "handleResult": "已通知护士查看，状态稳定",
  "handledBy": "nurse01"
}
```

### Response
- 返回更新后的预警状态

---

## 5. 发起云探视请求
### `POST /ops/remote-visits/request`

### Request Body
```json
{
  "elderId": 10001,
  "familyUserId": 20001,
  "requestedBy": "family",
  "remark": "想看看老人状态"
}
```

### Response
- 返回 `callId`

---

## 6. 接听云探视请求
### `POST /ops/remote-visits/{id}/accept`

### Request Body
```json
{
  "acceptedBy": "caregiver01"
}
```

---

## 7. 拒绝云探视请求
### `POST /ops/remote-visits/{id}/reject`

### Request Body
```json
{
  "rejectedBy": "caregiver01",
  "reason": "当前正在护理中，请稍后重试"
}
```

---

## 8. 结束通话
### `POST /ops/remote-visits/{id}/end`

### Request Body
```json
{
  "endedBy": "family",
  "durationSeconds": 300
}
```

---

## 9. 查询通话记录
### `GET /ops/remote-visits/my`

### Query 参数
- `page`
- `size`
- `elderId`
- `callStatus`

### 列表字段
- `callId`
- `elderId`
- `familyUserId`
- `callStatus`
- `startedAt`
- `endedAt`
- `durationSeconds`

---

## 10. 查询通知列表
### `GET /ops/notifications`

### Query 参数
- `page`
- `size`
- `readStatus`
- `notificationType`

### 列表字段
- `notificationId`
- `notificationType`
- `title`
- `content`
- `readStatus`
- `createdAt`

---

## 11. 标记通知已读
### `POST /ops/notifications/{id}/read`

---
