# 五、开始开发前建议先确认的文件级问题

1. `service-portal` 当前路由结构是否适合继续沿用 `/journey/*` 并按角色分流展示页面
2. `JourneyLayout.tsx` 是否容易改造成按角色显示不同菜单
3. `ServiceJourneyTaskService` 是否足够通用，能否不改名直接复用
4. 文件上传是否已有统一前端组件和后端接口
5. 是否要先在 `packages/common-react` 抽公共组件，还是阶段一先各写各的
